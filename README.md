## MMG Katsayı Tahmin Etme Aracı

Proje gemi manevra verilerinden MMG modelinin proje içinde belirtilen katsayılarını tahmin etmeye yarayan arayüz.

Kullanıcılar yan panelde bulunan gemiye ait bilgiler ve model ayarlamalarını yaptıktan sonra kendi veri setlerini yükleyerek 
Lineer Regresyon, LineerSVR ve Fizik Bildirimli Sinir Ağı(PINN) üzerinden tahminler yapabilir, model performans değerlerini ve 
model tahminlerini indirebilirler.
